Introduction
============

Pykrieg is a Pythonic wargame engine for Guy Debord's *Le Jeu de la Guerre*.

This project is currently in early development.

Quick Start
-----------

Installation::

   pip install pykrieg

Basic usage::

   import pykrieg
   
   print(pykrieg.placeholder())

For more information, see the GitHub repository: https://github.com/ssjmarx/pykrieg
